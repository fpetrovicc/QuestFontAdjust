# QuestFontAdjust (Turtle WoW / Vanilla 1.12)
A lightweight WoW addon for the Turtle WoW server that allows users to scale the quest font size without breaking the UI.

## Features
- Smart font detection (compatible with pfUI).
- Minimalist, draggable configuration UI.
- Range checking to prevent text overflow.
- Persists across game sessions.

## Usage
Type `/qfa` in-game to toggle the UI.
